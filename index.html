About

  • Think of this as a gym session for your focus muscle.
  • It's a todo list with timer and background brown noise.
  • Each succseful focus session is a rep done for your focus muscle
  • Timer will remind you to bring the focus back to the task after each 1,2,3,5,8 minutes if successfully focused, or restart with 1 min if distracted.

Prep

  • Add todos to be done in the next time block (recommended 2h timeblock)
  • First task could be to make the prioritised todolist.
  • Put phone on silent and out of reach
  • Get enough water within reach.

How to Use

  • Rule 1: Start Brown Noise for background sound
  • Rule 2: Start timer
  • Rule 3: Start working on first task
  • Rule 4: When Timer rings (after 1,2,3,5,8,13... mins) - Check if you were able to focus or did you get distracted?
  • Rule 5: If you're still focused, well done! - Click Keep Griding - to start the next longer timer
  • Rule 6: If you got distracted, don't worry, do 5 pushups, and click Got Distracted to go back to 1min timer
  • Rule 7: Repeat until the end of your time block.
